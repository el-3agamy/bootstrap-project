# Portfolio Website

This is a personal portfolio website built with **Bootstrap**, **Typed.js**, and **Font Awesome**. The website showcases skills, projects, and contact details with smooth animations and responsive design.

## Features

- **Responsive Layout:** Built with **Bootstrap** to ensure the website works seamlessly across devices of all screen sizes.
- **Animated Text:** Integrated **Typed.js** for typing animations on key elements to make the page more dynamic and engaging.
- **Icons:** **Font Awesome** is used for clean and attractive icons across the site, making it more visually appealing and interactive.
- **Modern Design:** Clean, minimalistic, and professional design that is easy to navigate.
- **Mobile-First Design:** Fully optimized for mobile devices, ensuring that it looks great on smartphones, tablets, and desktops.

## Technologies Used

- **HTML5:** For semantic markup and accessibility.
- **CSS3:** To style the page and provide modern visual effects.
- **Bootstrap:** A popular CSS framework for a responsive, mobile-first design.
- **Typed.js:** A JavaScript library for creating typing animations on the website.
- **Font Awesome:** A widely used icon library for including scalable vector icons.
